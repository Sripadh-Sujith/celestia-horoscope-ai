# 🔮 Celestia – Horoscope Expert AI

**Celestia** is an intelligent **Horoscope Expert AI** that provides personalized astrological insights through natural language conversations.  
It is powered by the ultra-fast inference of **Groq** and the interactive AI interface framework **Chainlit**.

Celestia acts like a professional astrologer, answering questions about life, career, love, and the future.

---

# ✨ Features

- 🔮 **Personalized Horoscope Predictions**  
  Get astrology insights based on your **name, birth date, and zodiac sign**.

- ⚡ **Ultra-Fast AI Responses**  
  Powered by **Groq API** for extremely fast AI inference.

- 💬 **Interactive Chat Interface**  
  Built with **Chainlit** for a clean and modern chat UI.

- 🧠 **Expert Astrology Prompting**  
  AI behaves like a **professional horoscope expert**.

- 🌍 **Simple Deployment**  
  Easily deployable on:
  - Hugging Face Spaces
  - Local server
  - Cloud platforms

---

# 🧠 How Celestia Works

1. User enters:
   - Name  
   - Date of Birth  
   - Zodiac Sign  
   - Horoscope Question  

2. The request is sent to the **Groq LLM API**.

3. The AI processes the request using a **Horoscope Expert prompt**.

4. **Chainlit** displays the response in a chat interface.
